# Medication Reminder System 💊

A modern desktop application built with **Python** to help patients manage their medication schedules effectively.

## 🚀 Features
- **Modern UI:** Built using `CustomTkinter` for a sleek, dark-themed interface.
- **Automated Alerts:** Integrated with the `schedule` library to trigger reminders at exact times.
- **System Notifications:** Sends Windows desktop notifications using `plyer`.
- **Smart Snooze (Zannan):** If a dose is skipped, the system re-alerts the user every 30 seconds until confirmed.
- **Logging System:** Automatically records whether a medication was "Taken" or "Skipped" in a local report file.

## 🛠️ Built With
- **Python 3.12**
- **CustomTkinter** (UI Framework)
- **Schedule** (Task Scheduling)
- **Plyer** (Desktop Notifications)
